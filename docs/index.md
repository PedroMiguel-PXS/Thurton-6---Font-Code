# 🚀 Thurton Browser 

An open-source web rendering engine built with JavaScript. Thurton is designed for speed, transparency, and modularity.

---

## 📥 Download Source Code
To compile and run Thurton on your local machine (Linux, macOS, or Windows), click the button below to get the latest version:

[**Download Project (.zip)**](https://github.com/PedroMiguel-PXS/Thurton-6---Font-Code/archive/refs/heads/main.zip)

---

## 🛠️ How to Build and Run

Since Thurton is a web-based engine, you don't need a heavy compiler. Follow these steps to get started:

1. **Extract** the downloaded `.zip` file.
2. **Open** the folder in your preferred environment (like VS Code or VSCodium).
3. **Launch** the `index.html` file in any modern web browser to see the engine in action.

### System Requirements
- **OS:** Any (Linux, Windows, or Mac).
- **Engine:** Any modern browser with JavaScript support.
