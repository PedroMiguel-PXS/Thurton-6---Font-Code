// Features

Built different.
By design.

Unlike traditional browsers, Thurton exposes its internals so developers can understand, modify, and extend every layer of the rendering pipeline.

🔬
Transparent Engine

Every step of HTML parsing, style computation, and layout is visible. No hidden magic — just readable JavaScript you can audit.

⚡
Font-Code Renderer

Version 6 introduces the Font-Code rendering system, a novel approach to text layout that treats typography as a first-class citizen.

🧩
Modular Architecture

Swap out components, extend the parser, override the layout engine — Thurton is built to be taken apart and rebuilt.

🌐
Zero Dependencies

Runs in any modern browser with JavaScript support. No Node.js required to launch, no heavy runtimes, no vendor lock-in.

🐧
Linux First

Designed with Linux in mind. Works flawlessly on Debian, Ubuntu, Arch, and any distro that ships a modern browser.

🔓
MIT Licensed

Use it, fork it, sell it. The MIT license gives you full freedom to integrate Thurton into any project, personal or commercial.

// Get Started

Up and running
in 3 steps.

01
Clone or Download

Get the source code from GitHub or download the zip directly.

git clone https://github.com/
PedroMiguel-PXS/Thurton-6
02
Extract & Open

No build step needed. Just open the project folder in your editor.

unzip Thurton-6-main.zip
cd Thurton-6-main
03
Launch Browser

Open browser.html in any modern web browser to run Thurton.

xdg-open browser.html
# or double-click the file

Start building
your browser.

Thurton is free, open, and waiting for you.
